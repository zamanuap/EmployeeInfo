This project uses the following technology

1.Asp.net Core
2.Entity Framework Core
3.Ajax

The features of the project are
1.This basically demostrates a CRUD operation.
2.It can save and retrieve accent (unicode) data.
3.It implements both client and server side custom validation.
4.It uses Ajax to show the list of employees without refreshing the whole page.
5.It implements pagination.
6.It returns to the same page where the record was after deleting a record.  


To create and restore database 
1.Open a new query window in SSMC and create a database uning following command
CREATE DATABASE Employee_DB
2.Restore Employee_DB.bak file to Employee_DB database.
